sphinx_test.app¶

sphinx_test.app.hello(name='keisatou')¶

say hello to name

パラメタ:name (str) – name of person to say hello to. Default is “keisatou”.
戻り値:None
戻り値の型:None

For example:

>>> import sphinx_test.app
>>> sphinx_test.app.hello()
hello, keisatou!
>>> sphinx_test.app.hello('world')
hello, world!
